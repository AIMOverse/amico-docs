# Packages Overview

## Core Crates

- [amico-core](/docs/packages/amico-core) - Defines the core components and interfaces for Amico. It does not implement any specific functionality, but only provides the core workflow and interface for building plugins.
- [amico-plugins](/docs/packages/amico-plugins) - A collection of plugins that actually implement the functionality of Amico. These plugins provide additional functionality such as event triggers, knowledge acquisition, decision making, and execution.
- [amico-std](/docs/packages/amico-std) - A collection of plugins that implements common functionality for Amico agents and provides basic functionality for building custom agents.
- [amico-firmware](/docs/packages/amico-firmware) - A crate that provides a low-level interface for interacting with embedded devices.
- [amico-macros](/docs/packages/amico-macros) - The crate for proc-macros.
