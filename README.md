# Amico Docs

This site is built using [Hugo](https://gohugo.io/) and [Hugo Theme Monochrome](https://github.com/kaiiiz/hugo-theme-monochrome).

## Quick Start

**Install Hugo**

You need "extended" version of hugo to support SASS. Checkout [Install Hugo](https://gohugo.io/getting-started/installing/) for more details.

**Update Git Submodules**

```bash
git submodule update --init --recursive
```

**Develop**

```bash
hugo server
```
